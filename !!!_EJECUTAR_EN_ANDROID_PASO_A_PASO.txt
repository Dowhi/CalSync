  📱 CÓMO EJECUTAR CALSYNC EN ANDROID STUDIO - PASO A PASO
═══════════════════════════════════════════════════════════════════════════════

🎯 PROBLEMA RESUELTO: Configuración para Android nativo

═══════════════════════════════════════════════════════════════════════════════
  ✅ PASO 1: INICIAR SERVIDOR DE DESARROLLO
═══════════════════════════════════════════════════════════════════════════════

En una terminal PowerShell, ejecuta:

   npm run dev

Espera a que diga: "Local: http://localhost:3000"

⚠️ NO CIERRES ESTA TERMINAL (debe seguir corriendo)

═══════════════════════════════════════════════════════════════════════════════
  ✅ PASO 2: ABRIR ANDROID STUDIO
═══════════════════════════════════════════════════════════════════════════════

En OTRA terminal PowerShell (deja la primera corriendo), ejecuta:

   npx cap open android

Esto abrirá Android Studio automáticamente.

═══════════════════════════════════════════════════════════════════════════════
  ✅ PASO 3: EJECUTAR EN ANDROID STUDIO
═══════════════════════════════════════════════════════════════════════════════

En Android Studio:

1. Espera a que sincronice Gradle (puede tardar 1-2 minutos)
   
2. Selecciona un emulador o conecta un dispositivo físico:
   - Si usas emulador: Click en el dropdown arriba
   - Si usas dispositivo: Conéctalo por USB y activa "Depuración USB"

3. Click en el botón verde ▶ "Run" (Play)

4. La app se instalará y abrirá en el dispositivo/emulador

═══════════════════════════════════════════════════════════════════════════════
  🐛 SI VES ERRORES DE RED EN ANDROID
═══════════════════════════════════════════════════════════════════════════════

Si el emulador no puede conectarse a localhost:3000:

1. En el emulador Android, ve a Configuración → WLAN
2. Anota la IP de tu computadora (ej: 192.168.1.100)
3. Modifica capacitor.config.ts:
   
   url: 'http://TU_IP:3000'
   
   (reemplaza TU_IP con la IP real)

4. Ejecuta: npx cap sync android
5. Ejecuta de nuevo en Android Studio

═══════════════════════════════════════════════════════════════════════════════
  📋 COMANDOS RÁPIDOS
═══════════════════════════════════════════════════════════════════════════════

# Terminal 1: Servidor de desarrollo
npm run dev

# Terminal 2: Abrir Android Studio
npx cap open android

# Después de cambios en el código:
npm run build          # Compilar
npx cap sync android   # Sincronizar con Android
                       # Luego ejecutar de nuevo en Android Studio

═══════════════════════════════════════════════════════════════════════════════
  💡 CONSEJO: PRUEBA PRIMERO EN NAVEGADOR
═══════════════════════════════════════════════════════════════════════════════

Antes de ejecutar en Android Studio, prueba en el navegador:

1. Ejecuta: npm run dev
2. Abre: http://localhost:3000
3. Abre DevTools (F12) → Modo móvil (Ctrl + Shift + M)
4. Selecciona dispositivo Android
5. Prueba toda la funcionalidad

Si funciona en el navegador, funcionará en Android Studio.
