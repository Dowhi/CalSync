═══════════════════════════════════════════════════════════════════════

⚠️  IMPORTANTE - ANTES DE EMPEZAR:

  1. Necesitas Node.js 18+ instalado
  2. Necesitas una cuenta de Google (para Firebase)
  3. El archivo .env DEBE configurarse con tus credenciales
  4. Las reglas de Firestore DEBEN publicarse en Firebase Console
  5. Para PWA: Necesitas generar iconos (ver ICONS.md)

═══════════════════════════════════════════════════════════════════════

🐛 ¿PROBLEMAS?

  ❌ Firebase errors         → Verifica .env con tus credenciales
  ❌ Permission denied       → Verifica reglas de Firestore
  ❌ No se instala PWA       → Usa HTTPS o localhost
  ❌ No se sincronizan datos → Verifica conexión + autenticación

  📖 Consulta INSTALL.md para solución de problemas comunes
