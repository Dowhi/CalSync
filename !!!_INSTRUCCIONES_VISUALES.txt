🎨 ELEMENTOS INTERACTIVOS:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

En LISTA DE TURNOS:
  ▼ (arriba derecha)    → Expande/contrae botones de acción
  [CREAR TURNO NUEVO]   → Abre configuración para nuevo turno
  [IMPORTAR TURNOS...] → Carga 7 turnos predefinidos
  [Turno]              → Click para editar
  ☰ (derecha)          → Menú contextual (Editar/Eliminar)

En CONFIGURACIÓN:
  [Nombre]             → Campo de texto editable
  [ASPECTO]            → Tab de aspecto visual
  [HORARIOS]           → Tab de configuración de horarios
  [ACCIONES]           → Tab de acciones adicionales
  [Color]              → Click para seleccionar color
  [Slider]             → Arrastra para cambiar tamaño
  [CANCELAR]           → Descartar cambios
  [GUARDAR]            → Guardar y volver
