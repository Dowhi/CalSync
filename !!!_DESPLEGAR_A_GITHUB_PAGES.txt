OPCIÓN 1: Deploy automático con Firebase Hosting
─────────────────────────────────────────────────

1. Construir para producción:
   npm run build

2. Desplegar a Firebase Hosting:
   firebase deploy --only hosting

3. Tu app estará en:
   https://calendario-sync-9e37f.firebaseapp.com


OPCIÓN 2: Deploy a GitHub Pages
────────────────────────────────

1. Construir para GitHub Pages:
   npm run build:github

2. Hacer commit de los cambios:
   git add .
   git commit -m "feat: Agregar funcionalidad de gestión de turnos"

3. Push a GitHub:
   git push origin main

4. GitHub Actions desplegará automáticamente

5. Tu app estará en:
   https://[tu-usuario].github.io/CalSync/


OPCIÓN 3: Deploy completo (Firebase + GitHub)
──────────────────────────────────────────────

1. Build para ambos:
   npm run build:github

2. Deploy a Firebase:
   firebase deploy

3. Commit y push a GitHub:
   git add .
   git commit -m "feat: Agregar funcionalidad de gestión de turnos"
   git push origin main


📱 VERIFICAR EL DEPLOY:
