🐛 ERRORES COMUNES Y SOLUCIONES:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

ERROR: "Cannot resolve module"
SOLUCIÓN: Problema con imports, ejecutar:
  npm install
  npm run build

ERROR: "Firebase: Permission denied"
SOLUCIÓN: Las reglas de Firestore, ejecutar:
  firebase deploy --only firestore:rules

ERROR: "Module not found: @/components/user/UserSelector"
SOLUCIÓN: Problema con alias de path, verificar vite.config.ts

ERROR: "Cannot read property 'map' of undefined"
SOLUCIÓN: PREDEFINED_USERS no se está cargando

ERROR: "Loading..." infinito
SOLUCIÓN: El hook useAuth no está funcionando correctamente
