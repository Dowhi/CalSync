⚠️ IMPORTANTE:

No olvides configurar Firebase:

1. Crea proyecto en Firebase
2. Habilita Firestore
3. Agrega las credenciales en:
   - Variables de entorno de GitHub (para producción)
   - O en .env (para desarrollo local)

Para producción en GitHub Pages:

1. Ve a: https://github.com/Dowhi/CalSync/settings/secrets/actions
2. Clic en "New repository secret"
3. Agrega estas variables:
   - VITE_FIREBASE_API_KEY
   - VITE_FIREBASE_AUTH_DOMAIN
   - VITE_FIREBASE_PROJECT_ID
   - VITE_FIREBASE_STORAGE_BUCKET
   - VITE_FIREBASE_MESSAGING_SENDER_ID
   - VITE_FIREBASE_APP_ID
