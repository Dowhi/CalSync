🔥 CONFIGURAR FIREBASE (IMPORTANTE):

Para que los eventos funcionen, necesitas Firebase:

OPCIÓN 1: Variables en .env (Local)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Ya tienes el archivo .env
2. Reemplaza con TUS credenciales de Firebase:

VITE_FIREBASE_API_KEY=tu_api_key_real
VITE_FIREBASE_AUTH_DOMAIN=tu_proyecto.firebaseapp.com
VITE_FIREBASE_PROJECT_ID=tu_proyecto_id
VITE_FIREBASE_STORAGE_BUCKET=tu_proyecto.appspot.com
VITE_FIREBASE_MESSAGING_SENDER_ID=123456789
VITE_FIREBASE_APP_ID=1:123456789:web:abc123

OPCIÓN 2: Variables en GitHub (Producción)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Para que funcione en GitHub Pages:

1. Ve a: https://github.com/Dowhi/CalSync/settings/secrets/actions

2. Clic en "New repository secret"

3. Agrega estos 6 secrets (uno por uno):

   Name: VITE_FIREBASE_API_KEY
   Secret: [tu api key]

   Name: VITE_FIREBASE_AUTH_DOMAIN
   Secret: [tu proyecto.firebaseapp.com]

   Name: VITE_FIREBASE_PROJECT_ID
   Secret: [tu proyecto id]

   Name: VITE_FIREBASE_STORAGE_BUCKET
   Secret: [tu proyecto.appspot.com]

   Name: VITE_FIREBASE_MESSAGING_SENDER_ID
   Secret: [tu sender id]

   Name: VITE_FIREBASE_APP_ID
   Secret: [tu app id]

4. Modifica .github/workflows/deploy.yml para usar los secrets

═══════════════════════════════════════════════════════════════════════

📋 CÓMO OBTENER CREDENCIALES DE FIREBASE:

1. Ve a: https://console.firebase.google.com/

2. Crea un proyecto nuevo (o usa uno existente)

3. Habilita Firestore:
   - Build → Firestore Database
   - Crear base de datos
   - Modo: Producción o Prueba

4. Obtén las credenciales:
   - Configuración del proyecto (⚙️)
   - Tus apps → Web (</> ícono)
   - Registra una app: "CalSync"
   - Copia el firebaseConfig

5. Pega las credenciales en .env o GitHub Secrets
