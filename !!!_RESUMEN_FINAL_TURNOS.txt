🎯 CÓMO USAR LA NUEVA FUNCIONALIDAD:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

PASO 1: Abre tu navegador
   👉 http://localhost:3000

PASO 2: Selecciona un usuario (si no estás logueado)

PASO 3: En la pantalla del Calendario, mira la barra inferior

PASO 4: Pulsa el botón "TURNOS" (a la derecha)
   ➡️ Se abrirá la página de gestión de turnos

PASO 5: En la pantalla de TURNOS DISPONIBLES
   - Pulsa el botón ▼ (arriba a la derecha) para expandir
   - Pulsa "CREAR TURNO NUEVO" para crear uno desde cero
   - O pulsa "IMPORTAR TURNOS..." para cargar los 7 turnos predefinidos

PASO 6: Para EDITAR un turno
   - Pulsa sobre cualquier turno de la lista
   - Se abrirá la pantalla de CONFIGURACIÓN DE TURNOS
   - Modifica:
     * Nombre del turno
     * Abreviatura
     * Color de fondo (paleta de 10 colores)
     * Color de texto (paleta de 10 colores)
     * Tamaño del texto (8-24)
     * Horarios de inicio y fin
   - Pulsa "GUARDAR" para aplicar cambios
   - O "CANCELAR" para descartar

PASO 7: Para ELIMINAR un turno
   - Pulsa el icono ☰ (menú) a la derecha del turno
   - Selecciona "Eliminar"
   - Confirma la acción

📱 PROBAR EN MÓVIL:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Conecta el móvil a la misma red WiFi que tu PC

2. Encuentra tu IP local:
   ipconfig
   (busca "Dirección IPv4", ej: 192.168.1.100)

3. En el móvil abre:
   http://[TU_IP]:3000

4. Instala como PWA:
   Chrome: Menú > Agregar a pantalla de inicio
